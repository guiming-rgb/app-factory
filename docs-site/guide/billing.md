## 如何使用

### 升级方案

1. 登录 Web 控制台
2. 点击右上角头像 → **订阅管理**
3. 选择目标方案
4. 选择支付周期（月度/年度）
5. 完成支付

年度付费享 **8 折优惠**。

### API 方式

```bash
# 获取方案列表
GET /api/billing/plans

# 订阅方案
POST /api/billing/subscribe
Content-Type: application/json

{
  "planId": "pro",
  "interval": "monthly",
  "successUrl": "https://your-app.com/billing/success",
  "cancelUrl": "https://your-app.com/billing/cancel"
}

# 查看用量
GET /api/billing/usage?workspaceId=xxx
```

### 查看用量

在 Web 控制台的 **用量仪表盘** 中可以实时查看：

- 本月代码生成次数
- AI 调用次数和 Token 消耗
- 存储使用量
- 团队成员数

## 计费 FAQ

### 如何升级/降级方案？

可以在"订阅管理"页面随时更改方案。升级即时生效，降级在当前计费周期结束后生效。

### 有没有年度折扣？

年度付费享 8 折优惠。Pro 年度 ¥2,870（节省 ¥718）。

### 免费方案有隐藏限制吗？

免费方案功能完整，仅在使用量上有上限。不会限制生成代码的质量或功能。

### 已生成的代码是否需要持续订阅？

不需要。生成的代码属于你，可以永久使用、修改和发布，无论订阅状态如何。

### 超出使用限制怎么办？

- **超出项目数** — 无法创建新项目，现有项目不受影响
- **超出代码生成次数** — 可购买额外次数包，或升级到更高方案
- **超出团队人数** — 需要移除成员或升级方案

### 支持哪些支付方式？

- 信用卡（Visa / Mastercard / American Express）
- 支付宝
- 微信支付

### 企业方案的定制模板如何收费？

定制模板包含在 Enterprise 方案中。如需开发行业专属模板，请联系客服。

### 如何取消订阅？

在"订阅管理"页面点击"取消订阅"。取消后，当前计费周期结束时方案降级为 Free。

### 取消后数据会丢失吗？

不会。你的项目和生成的代码会保留。但如果超过 Free 方案的限制（如 3 个项目），你将无法创建新项目。重新订阅后恢复所有功能。

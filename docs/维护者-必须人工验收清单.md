# 维护者 — 必须您亲自跑的验收清单

> **原则**：下列项 Agent **不能代替**（浏览器 OAuth、微信开发者工具、本机双进程长跑、生产网络）。  
> **状态真源**：[`maintainer-pending-state.json`](./maintainer-pending-state.json)  
> **每日提醒**：安装后每天 **5 次** 系统通知 → 见文末「每日 5 次提醒」

---

## 怎么用

| 操作 | 命令 |
|------|------|
| 看待办 | `npm run maintainer:pending` |
| 标完成 | `npm run maintainer:done -- M-S6-01 "verify:s6:local-full 通过"` |
| 手动提醒 | `npm run maintainer:remind` |
| 安装每天 5 次提醒 | `npm run maintainer:install-reminders` |

完成某项后**必须**跑对应 **verify** 或按「通过标准」自证，再 `maintainer:done`。


## 五、每日 5 次提醒（macOS）

在 **app-factory** 目录执行一次：

```bash
npm run maintainer:install-reminders
```

默认北京时间 **09:00、12:00、15:00、18:00、21:00** 弹出系统通知，并打印待办摘要。

| 管理 | 命令 |
|------|------|
| 查看已安装的 cron | `crontab -l \| grep maintainer-daily-remind` |
| 卸载 | `npm run maintainer:uninstall-reminders` |

**说明**：需允许「终端」通知；若 cron 不触发，可在 Cursor 里说「跑 maintainer:remind」补一次。

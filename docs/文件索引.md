## 概览

本项目是一个 **多平台 App 生产工厂** — Next.js 全栈 Web 应用，核心能力为：用户输入 App 描述 → AI 生成 Spec → 代码生成 Flutter / 微信小程序 / 鸿蒙应用 → GitHub 推送 + 桌面端 GHA 构建。

- **框架**: Next.js (App Router) + TypeScript
- **后端**: Supabase (Auth / Postgres / RLS) + Inngest (异步队列)
- **前端**: React + Tailwind CSS
- **模板生成**: Flutter / 鸿蒙 (HarmonyOS) / 微信小程序
- **LLM**: Claude API (spec 解析、代码生成、质量诊断)
- **沙箱**: Docker (Flutter 编译)

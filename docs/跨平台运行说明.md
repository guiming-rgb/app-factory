# 跨平台运行说明（工厂 + 生成 App）

> **目标**：工厂在 **Mac / Windows** 上可用；生成的 App 能在 **Mac 桌面、Windows 桌面、鸿蒙手机/平板、iOS/Android、微信** 对应环境中运行。

---

## 一、App 生产工厂（本平台）

| 环境 | 使用方式 |
|------|----------|
| **Mac 笔记本 / iMac 一体机** | 浏览器 https://app-factory-five.vercel.app · 本地 `npm run start -- -p 3001` |
| **Windows** | 同上；`git push` 常用代理 `127.0.0.1:7897` |

工厂是 **Web 应用**，不绑定单一操作系统。

---

## 二、生成 App 运行矩阵（按栈）

| 栈 | 运行环境 | Mac 电脑 | Windows 电脑 | 鸿蒙手机/平板 |
|----|----------|----------|--------------|---------------|
| **Flutter** | 原生安装 | ✅ `flutter run -d macos` | ✅ `flutter run -d windows` | —（用鸿蒙栈） |
| **鸿蒙 ArkTS ZIP** | 鸿蒙系统 | ❌ 非 Mac 应用 | ❌ 非 Win 应用 | ✅ **DevEco 安装运行** |
| **微信小程序** | 微信内 | ⚠️ 微信 Mac 版 | ⚠️ 微信 PC 版 | ⚠️ 部分机型微信鸿蒙版 |

说明：**「在鸿蒙上运行」= 生成的鸿蒙工程装到鸿蒙设备**（或 DevEco 模拟器），不是在 Mac/Windows 里直接双击鸿蒙 ZIP。

---

## 三、Flutter（Mac + Windows + iOS + Android）

工程含 `macos/`、`windows/`、`ios/`、`android/`。

```bash
flutter pub get
flutter run -d macos      # 苹果 Mac
flutter run -d windows    # Windows
flutter run -d chrome     # 可选 Web 预览（非商店上架主路径）
```

### 可双击发行包（无需最终用户装 Flutter）

见 [桌面可双击发行包.md](./桌面可双击发行包.md)。工厂在 **Mac 构建机** 可产出 `*-macos.app.zip`，在 **Windows 11 构建机** 可产出 `*-windows.zip`；或使用 GitHub Actions **Flutter 桌面双平台构建** 一次拿齐。

`targets.flutter.platforms` 默认：

```json
["ios", "android", "macos", "windows"]
```

## 四、鸿蒙（HarmonyOS）— 在鸿蒙系统上安装运行

### 工厂已具备

- 同步生成：**「生成鸿蒙 ZIP（同步）」**（约 10～15 秒）
- 模板 + Codegen：列表、**详情页**、`loadItems` + **Supabase REST**
- 权限：`INTERNET`（`module.json5`）
- 探针：`npm run verify:c6:harmony` · `verify:g2` · `verify:p1:production:sync`（harmony）

### 你在鸿蒙设备上的步骤

1. 生产站项目页 → **生成鸿蒙 ZIP（同步）** → 下载 ZIP  
2. 解压（根目录含 `entry`、`AppScope`）  
3. 安装 **DevEco Studio**（Mac 或 Windows 上开发，**运行在鸿蒙设备**）  
4. **File → Open** 选解压后的工程根目录（不是工厂里的 `templates/...`）  
5. 连接鸿蒙手机或启动鸿蒙模拟器 → **Run**  
6. 列表点击条目可进 **EntityDetail** 详情页（批次 O 已支持）

### App Spec 默认

```json
"harmony": { "enabled": true, "formFactors": ["phone", "tablet"] }
```

新项目 PoC Spec、Report→Spec 合并后 **默认开启鸿蒙**，与 Flutter / 小程序并列三栈。

### 验收（无需真机也可部分完成）

```bash
npm run verify:c6:harmony
npm run verify:p1:production:sync   # 仅 harmony
# 或三栈
npm run verify:p1:production:sync:all
```

解压 ZIP 检查：`Index.ets` 含 `SUPABASE_URL`、`router.pushUrl`、`EntityDetail.ets` 存在。

## 五、微信小程序

在 **微信客户端** 内运行（非独立 Mac/Win 安装包）。  
Windows / Mac 可用 **微信开发者工具** 导入 ZIP 调试。

---

## 六、验收命令汇总

| 命令 | 作用 |
|------|------|
| `npm run verify:p:desktop:flutter` | Flutter 含 macOS/Windows 目录 |
| `npm run verify:c6:harmony` | 鸿蒙模板 + 生成 + 待办/列表探针 |
| `npm run verify:p1:production:sync:all` | 生产三栈同步 codegen |

### 常用 API 路径（前缀 = 生产站或 localhost:3001）

| 用途 | 路径 |
|------|------|
| 生成 Flutter | `POST /api/projects/{projectId}/codegen/flutter` |
| 生成小程序 | `POST /api/projects/{projectId}/codegen/wechat` |
| 生成鸿蒙 | `POST /api/projects/{projectId}/codegen/harmony` |
| Codegen 记录 | `GET /api/projects/{projectId}/codegen/runs` |
| 下载源码 ZIP | `GET /api/projects/{projectId}/codegen/runs/{runId}/download` |
| Win 桌面包 | `…/download?kind=windows` |
| Mac（生产 302 跳 GitHub） | `…/download?kind=macos` |
| GitHub OAuth 回调 | https://app-factory-five.vercel.app/api/github/oauth/callback |

| 端 | 用户看到的名字 | 工程/包/目录名 |
|----|----------------|----------------|

### 本机路径示例（维护者实测）

| 端 | 路径 |
|----|------|
| 小程序（微信开发者工具导入） | `/Users/guiming/Downloads/simple_todo-wechat` |
| Mac `.app` | 解压 GHA `macos-{run_id}.zip` → **`app_factory_minimal.app`** |
| Windows | 解压 `windows-{run_id}.zip` → Release 内 `.exe` |
| 鸿蒙 | 解压 ZIP → **`simple_todo-harmony/`** 根目录（含 `entry/`、`AppScope/`） |

### GitHub 三栈镜像（S6 参考）

| 栈 | 仓库 |
|----|------|
| Flutter | https://github.com/guiming-rgb/local_todo-flutter |
| 微信小程序 | https://github.com/guiming-rgb/simple_todo-wechat |
| 鸿蒙 | https://github.com/guiming-rgb/simple_todo-harmony |

### 工厂内获取方式

| 端 | 入口 | 说明 |
|----|------|------|
| 源码 ZIP | 项目页 → 生成 → 源码 ZIP | `/api/.../download` |
| Win .exe 包 | 项目页 → **Win .exe 包** | `?kind=windows` |
| Mac .app | 项目页 → **Mac .app(GitHub)** | 跳 GHA Artifacts（Vercel 不代理大文件） |

### 各端运行环境

| 端 | 需要 |
|----|------|
| Mac 桌面 | 不需 Flutter；解压后 **右键打开** `.app`（GHA 未公证） |
| Windows | 不需 Flutter；双击 `.exe` |
| 微信小程序 | 微信开发者工具 + **测试号** AppID |
| 鸿蒙 | DevEco Studio + 鸿蒙设备/模拟器 |

# v2.1 — Flutter Docker 沙箱

> **状态**：已实现（2026-05-20）  
> **镜像**：`app-factory-flutter-sandbox:stable`（`docker/flutter-sandbox/Dockerfile`）

## 门禁

```bash
npm run sandbox:flutter:docker:analyze-only
```

流程：校验样例 Spec → 生成到 `/tmp/app-factory-sandbox-flutter-docker` → **容器内** `flutter pub get` + `dart analyze`。

完整含 APK build：

```bash
npm run sandbox:flutter:docker
```

重建镜像：`npm run sandbox:flutter:docker -- --rebuild-image`

## 命令组 G9-Docker

```bash
cd "/Users/guiming/Desktop/app生产工厂/app-factory"
npm run sandbox:flutter:docker:analyze-only
echo "G9-Docker 完成"
```

## 维护者

- 需安装并启动 **Docker Desktop**（Agent 无法代装）。
- 首次会 `docker build`（拉取 `ghcr.io/cirruslabs/flutter:stable`，可能较久）。

## 与本机 G9 关系

| 方式 | 命令 | 隔离 |
|------|------|------|
| 本机 | `sandbox:flutter:analyze-only` | 使用本机 Flutter |
| Docker | `sandbox:flutter:docker:analyze-only` | 工具链在容器内 |

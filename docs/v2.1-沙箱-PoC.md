## 门禁

`npm run sandbox:flutter`：

1. 校验样例 Spec  
2. 生成工程 → `/tmp/app-factory-sandbox-flutter`  
3. `flutter pub get` + `dart analyze`（**必过**）  
4. `flutter build apk --debug`（本机有 Flutter 时尝试；超时 5min）

跳过 build：`npm run sandbox:flutter -- --no-build`

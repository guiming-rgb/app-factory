# 记账模板 (Finance) — 对标随手记/挖财

## 能力
- 交易 CRUD（收入/支出、分类、金额）
- 月度统计（收入/支出汇总 + 分类饼图）
- 预算管理（分类预算 + 进度条）
- 多账户（现金/银行卡/支付宝/微信）
- 汇率转换（exchangerate-api.com 免费 API）
